L’équipe FindMe est honoré de vous presenter notre moteur de recherche !

Avant de pouvoir l’utiliser, veuillez suivre les instructions suivantes : 
- Créer un java projet sur éclipse (ou une autre IDE java)
- Entrez dans le répertoire eclipse-workspace, ensuite dans le répertoire du projet que       vous venez de créer et copier collez les répertoires contenus dans le répertoire Find Me.
- JUnit . Effectuez une clique droit sur le java projet et suivez le chemin “Build Path” —> “Add libreries…” —> JUnit —> Next —> JUnit 5 —> Finish
- Processing Effectuez une clique droit sur le fichier core.Jar, ensuite “Build Path” et “Add to Build Path”.  Si vous n’avez pas Processing sur votre machine, installez l'en suivant le lien : https://processing.org/download/

Vous pouvez maintenant lancer le logiciel FindMe avec le button “Run”. 

Bonne recherche, 
équipe FindMe
